Claude.ai — Setup & Interface Tour

Setup akun, kenali interface, dan mulai pakai Claude hari ini

Di modul ini kamu akan setup akun Claude, memahami perbedaan model (Haiku, Sonnet, Opus), dan langsung mencoba conversation pertama. Target: dalam 10 menit pertama kamu sudah punya akun dan bisa mulai chat.
📝 Coba Prompt Ini
Halo Claude, saya [nama] bekerja sebagai [jabatan] di [industri]. Saya baru pertama kali pakai Claude. Bisa bantu saya memahami 3 cara terbaik untuk memanfaatkan Claude dalam pekerjaan sehari-hari saya? Berikan contoh spesifik yang relevan dengan bidang saya.
🎯 Contoh: HRD Manager

"Halo Claude, saya Rina, HRD Manager di perusahaan manufaktur. Saya baru pertama kali pakai Claude. Bisa bantu saya memahami 3 cara terbaik memanfaatkan Claude untuk pekerjaan HR sehari-hari?"

Prompt Anatomy — Konteks + Instruksi + Format

Struktur prompt yang menghasilkan output tepat setiap kali

Formula prompt yang efektif selalu punya 3 bagian: Konteks (siapa kamu, apa situasinya), Instruksi (apa yang kamu minta), dan Format (bagaimana output yang kamu inginkan). Kuasai ini dan 80% output Claude akan langsung tepat.
📝 Template Prompt Anatomy
[KONTEKS] Saya adalah [jabatan] di [perusahaan/industri]. [SITUASI] Saya sedang mengerjakan [deskripsi pekerjaan/masalah]. [INSTRUKSI] Tolong bantu saya [apa yang diminta dengan spesifik]. [FORMAT] Sajikan dalam format [list/tabel/paragraf/poin] dengan panjang sekitar [pendek/sedang/detail]. [TAMBAHAN] Gunakan Bahasa Indonesia yang [formal/kasual/profesional].
🎯 Contoh Nyata: Manajer Proyek

"[KONTEKS] Saya Manajer Proyek di startup fintech. [SITUASI] Baru selesai meeting 1 jam dan punya 2 halaman catatan berantakan. [INSTRUKSI] Tolong buatkan ringkasan meeting dengan action items, PIC, dan deadline. [FORMAT] Gunakan format tabel, bahasa formal."

Role Prompting — Claude sebagai Ahli

Tingkatkan kualitas output dengan minta Claude berperan sebagai spesialis

Ketika kamu meminta Claude "berperan sebagai" ahli tertentu, output yang dihasilkan jauh lebih relevan dan tajam. Teknik ini sangat berguna untuk konsultasi, review, atau situasi yang butuh perspektif profesional.
📝 Template Role Prompting
Berperan sebagai [spesialis/ahli] berpengalaman 10+ tahun di bidang [bidang spesifik] untuk konteks Indonesia. Saya sedang menghadapi situasi berikut: [jelaskan situasi kamu] Berikan analisis dan rekomendasi konkret yang bisa saya jalankan hari ini.
🎯 5 Role yang Paling Berguna

Senior HRD Konsultan · Business Strategist · Senior Copywriter · Financial Analyst · Customer Service Expert. Coba satu role yang paling relevan dengan pekerjaan kamu hari ini.

Claude Artifacts — Generate Dokumen Langsung

Buat tabel, template, dan dokumen tanpa copy-paste ke tools lain

Claude Artifacts memungkinkan kamu membuat dokumen, tabel interaktif, dan template langsung di dalam Claude — hasilnya bisa di-download atau dibuka di tab terpisah. Tidak perlu buka Google Docs dulu.
📝 Prompt untuk Artifacts
Buatkan [jenis dokumen: tabel/template/checklist] untuk: [deskripsi apa yang kamu butuhkan] Gunakan format yang rapi dan langsung bisa dipakai. Sertakan semua kolom/bagian yang dibutuhkan untuk [tujuan spesifik dokumen].
🎯 Contoh: Guru SMA

"Buatkan 20 soal pilihan ganda Matematika kelas 10 topik Fungsi Kuadrat via Artifacts. Sertakan kunci jawaban di bagian bawah. Format: siap cetak."

Claude Projects — Simpan Konteks Kerja

Setup project pertama kamu agar tidak perlu jelaskan ulang tiap sesi

Claude Projects menyimpan konteks pekerjaan kamu — tone, SOP, informasi perusahaan, atau prompt library. Setiap kali kamu buka project, Claude sudah "tahu" siapa kamu dan apa yang kamu butuhkan.
📝 System Prompt untuk Project Pertamamu
Kamu adalah asisten profesional saya. Berikut konteks yang perlu kamu tahu: TENTANG SAYA: - Nama: [nama kamu] - Jabatan: [jabatan] - Perusahaan: [nama/industri] - Tim: [ukuran/struktur tim] GAYA KOMUNIKASI: - Selalu gunakan Bahasa Indonesia - Tone: [formal/semi-formal/casual] - Format default: [poin-poin/paragraf] TUGAS YANG SERING SAYA MINTA: - [list 3-5 tugas yang paling sering kamu kerjakan]
🎯 Tips: Buat 3 Project Berbeda

Project 1: Pekerjaan Utama · Project 2: Konten & Komunikasi · Project 3: Proyek Sampingan. Setiap project punya konteks dan tone yang berbeda.

Google Docs Export — Workflow End-to-End

Dari prompt Claude sampai Google Docs siap dikirim ke atasan

Workflow lengkap: input data/catatan → prompt ke Claude → output terformat → copy ke Google Docs → kirim. Modul ini mempraktikkan semua skill sebelumnya dalam satu workflow nyata yang bisa kamu pakai mulai besok.
📝 Prompt Workflow Lengkap
Berikut [catatan meeting/data/draft] yang perlu diformat: [tempel catatan/data kamu di sini] --- Tolong ubah menjadi [jenis dokumen] yang siap saya kirim ke [atasan/klien/tim]. Format: [struktur yang kamu inginkan] Panjang: [pendek/sedang/lengkap] Tone: [formal/profesional/kasual] Sertakan: [hal spesifik yang harus ada]
🎯 Challenge Akhir Modul

Ambil satu pekerjaan nyata dari hari ini — meeting notes, laporan, atau email — dan coba workflow lengkap ini. Output ke Google Docs dan kirim ke diri sendiri.
